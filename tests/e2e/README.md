# E2E Tests

End-to-end tests for the Web Presence application using Playwright.

## Quick Start

```bash
# Install dependencies
npm install

# Run all tests
npm run test:e2e

# Run tests with UI
npm run test:e2e:ui
```

## Prerequisites

- Node.js 18+
- Backend API running
- Frontend application accessible
- Chrome, Firefox, Safari (for cross-browser tests)
